## Running Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Optional: Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
4. Open the app at [localhost:3000](http://localhost:3000/)
